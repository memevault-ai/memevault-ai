# memevault-ai

Premium crypto intelligence terminal presented as:
- product demo
- live simulation
- architecture-ready scanner
- treasury rotation prototype

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Zustand
- Recharts
- Lucide React

## Routes
- `/`
- `/bot-live`
- `/treasury`
- `/distributions`
- `/docs`

## Local development
```bash
npm install
npm run dev
```

## Notes
All data and API routes are mocked so the repo is safe to demo, extend and push to GitHub immediately.
